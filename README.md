# Fronted-TFG-DAW
Este repositorio contiene el frontend de la plataforma educativa desarrollada como Proyecto Final de Ciclo (DAW). El frontend es la parte visual de la aplicación y se encarga de la interacción directa con el usuario final.

La aplicación ofrece una interfaz intuitiva y accesible para alumnos, profesores y administradores, permitiendo el acceso a contenidos educativos, la comunicación entre usuarios y la gestión de perfiles de profesores particulares. Todo el contenido mostrado en la aplicación se obtiene a través de una API REST proporcionada por el backend.

El frontend ha sido diseñado para ofrecer una experiencia de usuario clara y moderna, incorporando funcionalidades como navegación estructurada, buscador de contenidos, formularios de interacción y soporte para modo claro y modo oscuro.

Funcionalidades principales

Registro e inicio de sesión de usuarios

Visualización de contenidos educativos

Publicación y gestión de comentarios

Buscador de asignaturas y contenidos

Visualización de perfiles de profesores particulares

Mensajería interna entre usuarios

Panel de administración

Modo claro y modo oscuro

Diseño responsive

Experiencia de usuario

El diseño del frontend prioriza la usabilidad y la claridad, permitiendo que los usuarios encuentren fácilmente los contenidos y funcionalidades. La interfaz está pensada para adaptarse a distintos dispositivos y tamaños de pantalla, garantizando una experiencia consistente.

Comunicación con el backend

El frontend se comunica con el backend exclusivamente mediante peticiones a la API REST. Todas las operaciones (autenticación, obtención de datos, envío de mensajes, publicación de contenidos, etc.) se realizan a través de llamadas HTTP, manteniendo el sistema desacoplado y modular.
